# Animation 클래스

- m_fDuration : 현재 애니메이션 트랙의 총 길이
- m_fTickPerSecond : 현재 애니메이션 트랙의 초당 재생 속도
  |- TickPerSecond가 하나의 변수로만 관리될 때 애니메이션이 어색한 경우가 종종 있다.
  |- 특정 구간에서 느려지거나 빨라져야 하는 경우가 존재함. (자연스러운 모션을 위해서는 해당 변수를 직접 조작해야하는 경우도 있음)
  |- 역재생도 구현 가능할듯??
- m_iNumChannels : 애니메이션에서 컨트롤해야하는 Bone들의 개수을 의미한다.
  |- 하나의 뼈가 다양한 상태 정보를 가져야 하는 경우가 있다.
