# Animation

## Animation - Animation 사이의 전환 처리

- 도중에 다른 애니메이션으로 전환되는 것을 임의의 시간 값을 두고 처리
- 전체 뼈의 상태와 다음 Animation의 뼈 상태를 보간한다.
- 뼈:뼈 1:1 매칭이 아니라 전체 뼈 개념으로 보간을 해야하는듯

- 실제 애니메이션 재생했을 때 기존 위치에서 재생되는 것이 아니라 움직이며 재생되는 것은 Root 위치가 변경되는 것으로 볼 수 있는데
- 해당 위치를 World에 설정하는 메서드가 필요

- Animation을 하나만 들고 있지 않고 두 개를 세트로 들고 있을 경우 상 하체 분리 Animation 가능
