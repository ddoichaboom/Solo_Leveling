- 정점들이 시간에 따라 따라갈 기준점(뼈대)의 변화값을 저장

# 정점들에게 뼈대의 행렬을 곱해주는 행위는 셰이더(하드웨어)를 통해 연산을 처리해야한다.

- 뼈 행렬들을 셰이더로 던져야함
- 정점 구조체 안에다가 이 정점은 어떤 뼈대를 따라간다 라는 정보를 담게 함

# 스켈레톤 구조

- 뼈들 간에 상속의 개념을 가져간다.
- 자식 뼈가 부모 뼈를 따라가야하고, 자식 뼈의 행렬(크·자·이)를 곱한 후 부모 행렬을 곱하면 원점 기준이 아닌 부모 뼈 기준으로 붙는다.

## CBone 클래스

### Assimp에서 뼈를 표현하는 데이터의 종류

aiNode, aiBone, aiAnimMode

aiNode
부모가 누구인지 자식이 누구인지 연결된 정보

aiBone
몇개의 정점에게 영향을 주는가?

aiAniMode
특정 위치에서의 상태값을 시간대별로 표현해놓은 것

(1) 뼈의 상태정보를 표현
(1-1) 부모의 상태 포함

```
public:
    static CBone* Create(const aiNode* pAINode);

private:
    _char m_szName[MAX_PATH] = {};
    _int m_iParentIndex = { -1 };       // 부모가 누구인가를 저장
```

AINode에서 Transformation 행렬을 가져와도 열기준 행렬이라 전치해줘야 함
