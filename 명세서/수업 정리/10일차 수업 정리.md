# Material 클래스

- 어떤 Mesh가 어떤 Material을 사용하는지 (인덱스)정보가 안에 담겨 있음 AIMesh->m_MaterialIndex;
- 하나의 Material 안에는 다양한 정보가 담겨 있음 (Diffuse, Ambient, Specular 등)
- 해당 정보는 AI_TEXTURE_TYPE_MAX에 들어 있음(최대 27개 - Assimp 버전에 의존적).

## 멤버 변수

{
private:
vector<ID3D11ShdaerResourceView\*> m_Materials[AI_TEXTURE_TYPE_MAX];
}

## 한 Mesh 파츠의 Material 안의 Diffuse가 여러개 있을 수 있다

- 하나의 파츠의 색상을 여러개 갖게 해서 특정 상황을 구분하고자 할때
