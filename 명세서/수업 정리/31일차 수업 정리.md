# 후처리 셰이더의 결과물

마우스가 ViewPort상에 어디에 위치하는지 찾고 Depth를 통해 월드 위치를 찾을 수 있다?
Depth Target에는 원하지 않는 것들이 기록될 수도 있다.

아예 렌더 타겟을 월드 위치를 기록하는 타겟을 만들어서 사용하는 것도 가능

Picking을 위해 월드 위치를 담고 있는 렌더타겟용 텍스처는 동적으로 생성할 수 없다?

Picking 클래스 내부에 동적 텍스처를 두고 렌더 타겟에 월드 위치가 기록된 정적 텍스처를 복사해와서 해당 텍스처를 Picking에 사용한다.

한 프레임 오차가 생길 수 있음 Render Call 이후에 텍스처로 만들어지고 해당 월드 좌표를 가져오는 것은 다음 프레임의 Update에서 실행될 것이기 때문에
