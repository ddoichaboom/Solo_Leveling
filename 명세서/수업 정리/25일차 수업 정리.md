# AABB / OBB / SPHERE

- 우리는 충돌이 아닐 때를 찾아야함

## AABB

### 축별 비교

- X 값에 대해서 교차했는지
- 깊이 값에 대해서 교차했는지
- 높이 값에 대해서 교차했는지

### Min, Max 포인트

- Min
  : 박스 상에서 x,y,z가 제일 작은 지점
- Max
  : 박스 상에서 x,y,z가 제일 큰 지점

## OBB

- 기존 X,Y,Z에서 바라봤을 때 겹쳐있다고 나오는 경우
- 각 박스의 축에서 바라보며 비교 (다채로운 방향)
- 15개의 축에 대해서 충돌 검출
- 축끼리 외적해서 새로운 축을 만들어내 해당 축에서도 비교
- 약 6개 ~ 15개 정도 사용
- 비교 판단하고자 하는 축에 각 박스의 중심점으로부터 수직 선을 그었을 때
- 해당 선상에서 박스의 중심점으로부터 가장 먼 점까지의 길이(박스 두개) 둘을 더했을 때 두 선을 더한 길이 값과 해당 선상에 찍힌 두 점 사이의 거리를 비교
- 각 코너 정점,
- m_pDesc->GetCorners
- 15개 비교하는 로직 공부해보기
