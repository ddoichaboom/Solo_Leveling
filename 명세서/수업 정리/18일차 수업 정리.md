# Font 파일 만들기

- MakeSpriteFont - bin - .exe 파일
- cmd 실행 - MakeSpriteFont 입력

- Font를 그려내기 위해 세팅했던 RenderState가 적용되어버림
- RenderState를 각 셰이더마다 다 세팅해서 문제가 생기지않게 해야함
- 한글 출력이 안되는 현상
  - CharacterRegion 설정해줘야 다른 나라의 문자를 담을 수 있다.
- 크게 만들고 작게 그리는 방향으로 진행
