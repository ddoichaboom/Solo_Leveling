# 인스턴싱

## 개념

- 같은 모델을 게임 내에 다수 출력해야 할 때
  EX) 이펙트

- 셰이더의 Apply함수는 생각보다 오버헤드가 큰 함수

- 그래서 등장한 기법이 인스턴싱 기법

```
// m_pContext->DrawIndexed() 대신에
m_pContext->DrawIndexedInstanced() // 함수 사용
```
