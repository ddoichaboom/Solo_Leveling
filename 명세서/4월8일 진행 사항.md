## 완료 항목

### Step 1: Engine .bin 로더 마무리

- `CModel::Load_Binary_Desc` / `Free_Binary_Desc` 구현 완료
- `CModel::Create(device, context, pBinaryPath)` 오버로드 — Load → Initialize_Prototype → Free 순서
- DBG_NEW 매크로 충돌 해결: `::operator new` 대신 `new _ubyte[]` 패턴 확정
- SLMD v1 포맷 스펙 Model.cpp 상단 주석 블록에 박제

### Step 2: Editor Assimp 통합

- `Editor/Public/assimp/` — Assimp 5.x 헤더
- `Editor/ThirdPartyLib/assimp-vc143-mt.lib`, `assimp-vc143-mtd.lib`
- `Editor/Bin/assimp-vc143-mt.dll`, `assimp-vc143-mtd.dll`
- Editor.vcxproj 추가 포함 디렉터리 / 라이브러리 설정
- Engine/Client 무의존 유지

### Step 3: Editor Model_Converter 클래스

- `CModel_Converter` static 유틸리티 클래스 (`Editor/Public/Model_Converter.h`, `Editor/Private/Model_Converter.cpp`)
- `Convert(pFbxPath, pBinPath, eModelType)` — Assimp 로드 → SLMD v1 .bin 쓰기
- `Collect_Bones(aiNode*, iParentIndex)` — DFS 순서로 본 노드 수집
- `Write_Meshes` — NONANIM(VTXMESH)/ANIM(VTXANIMMESH) 분기, 본 가중치 채우기, 0-bone 폴백
- `Write_Materials` — aiTextureType 순회, DIFFUSE 없을 때 _CO 텍스처 자동 탐색
- `Write_Bones` — Assimp 열우선 → DirectX 행우선 전치
- `Write_Animations` — KEYFRAME POD 배열 일괄 fwrite
- `Should_Skip_Mesh` — LowMesh / _NLOD(!=0) 필터링
- `Build_TexturePath` — Assimp 텍스처 경로 → `../../Resources/...` 상대 경로 변환

### Step 4: FBX 변환 트리거 (Content Browser 통합)

- 계획: MenuBar "File → Convert FBX..." 메뉴
- **실제 구현: Content Browser 컨텍스트 메뉴 방식으로 변경** (더 직관적인 UX)
- .fbx 파일 우클릭 → "Convert to .bin" → 모달 팝업
- 팝업: FBX/BIN 경로 표시, NONANIM/ANIM RadioButton 선택, Convert/Cancel 버튼
- 변환 성공/실패 Log 패널 출력 + Content Browser 자동 새로고침
- `Panel_ContentBrowser.h`에 멤버 추가: `m_FbxToConvertPath`, `m_bOpenConvertPopup`, `m_iModelType`
- `Render_ConvertPopup()` 메서드 추가

### Step 6: Viewport 모델 렌더 + Shader_VtxAnimMesh (Step 5보다 먼저 구현)

**Shader_VtxAnimMesh.hlsl:**
- VS_IN: POSITION, NORMAL, TEXCOORD, TANGENT, BINORMAL, BLENDINDEX(uint4), BLENDWEIGHT(float4)
- 4본 스키닝: `BoneMatrix = g_BoneMatrices[idx] * weight` 가중합
- Phong 라이팅: Diffuse + Ambient + Specular
- `g_BoneMatrices[512]`, `g_fAlpha` 변수

**CModelObject (Client DLL):**
- `MODELOBJECT_DESC` — ShaderProtoTag + ModelProtoTag (문자열 포인터)
- Initialize: __super::Initialize → Ready_Components (Com_Shader + Com_Model 프로토타입 Clone)
- Update: ANIM 타입이면 `m_pModelCom->Play_Animation(fTimeDelta)`
- Late_Update: NONBLEND 렌더 그룹 등록
- Render: 메시별 루프 — BoneMatrices 바인딩 → Material 바인딩 → Shader Begin → Model Render
- Bind_ShaderResources: WVP 행렬 + 카메라 위치 + 라이트 정보

**EditorApp::Ready_TestScene() 확장:**
- Shader_VtxMesh / Shader_VtxAnimMesh 프로토타입 등록
- SungJinWoo_ERank .bin 모델 프로토타입 등록 (CModel::Create 바이너리 경로 오버로드)
- CModelObject 프로토타입 + 인스턴스 배치

**변환된 .bin 파일 (4개):**
- `Resources/Models/SungJinWoo_ERank/SungJinWoo_ERank (merge).bin`
- `Resources/Models/Fiona/Fiona.bin`
